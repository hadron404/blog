## 必要条件

1. 安装 Git
2. 安装 Nodejs
3. GitHub



### 步骤

#### 安装

```bash
npm install hexo-cli
```

#### 需要本地运行时安装

```bash
npm install hexo-server --save
```



### 初始化

```bash
hexo init <your blog stored folder>
```



### 本地运行

```bash
hexo server
```

默认浏览URL：`127.0.0.1:4000`



### 部署

```bash
hexo deploy 
```
